# India Crime Analysis Dashboard

An interactive data visualization project that analyzes crime patterns across India. This dashboard provides insights into crime trends, victim demographics, officer performance, and case status using advanced visual analytics. The project is designed for data analysis, decision support, and portfolio demonstration.

---

## Dashboard Overview

The dashboard presents a comprehensive analysis of crime data with the following key metrics:

- Total Crimes: 600  
- Solved Cases: 317  
- Open Cases: 283  
- Crime Clearance Rate: 52.83%  

It helps users understand crime distribution, identify high-risk areas, and evaluate law enforcement performance.

---

## Key Features

### Crime Analysis
- Crime distribution by type (Robbery, Cyber Fraud, Drug Trafficking, Assault, etc.)
- State-wise crime visualization using an interactive map
- Monthly and yearly crime trends
- Comparison of total crimes vs solved crimes

### Victim Analysis
- Total victims overview
- Gender distribution (Male vs Female)
- Victims by age group:
  - Minor
  - Young Adult
  - Adult
  - Senior
- Solved cases by age group
- Risk level identification

### Officer Performance
- Cases assigned to individual officers
- Experience level analysis
- State-wise officer distribution
- Solved crimes by rank (DCP, SP, ACP, Inspector, SI)

### Crime Relationship Tree
Interactive hierarchical analysis based on:
- State  
- Crime Type  
- Age Group  
- Gender  
- Case Status  

This allows multi-dimensional exploration of crime patterns.
